# Sample ToDo App

### Tech Stack

- **Language**: Rust
- **Framework**: Tide
- **Database**: SurrealDB

### How to run?
1. Copy env file and set variables
```bash
cp .sample.env .env
```

2. Run with Docker
```bash
docker-compose up
```
